# Roll-A-Ball
[Unity Roll-A-Ball](https://learn.unity.com/project/roll-a-ball-jp?language=ja)

## 目的
Unityの初心者向けプロジェクトをC++で作成すること

![Screen shot](https://github.com/gamusiro/Roll-A-Ball/blob/master/screenshots/Game.png)

## 方針
- enttライブラリを使用したECS設計への意識(中途半端)
- OpenGLでのレンダリング処理
- 入力制御に関してはKeyboardのみでの対応(Mouseも追加済み)
- オーディオ系の処理は除く

## 利用ライブラリ
- glfw
- glad
- glm
- entt
- bullet3
- spdlog
- assimp
- stb
- utf
- freetype

## 対象
Windows
